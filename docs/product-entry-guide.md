# 帝昂官网产品添加说明

适用范围：供应商产品表格、产品 PPT、PDF 选型手册、证书资料、产品图片包。

## 1. 你需要提供什么

每次新增产品时，可以直接把以下任意资料发给 Codex：

- 供应商 Excel 表格
- 产品 PPT
- 产品 PDF 手册
- 产品图片或宣传图
- 型号清单
- 证书、检测报告、授权资料
- 你希望重点展示的产品名单

如果资料很多，建议按供应商或产品线分批发送，例如：

- MSA 气体检测仪一批
- 消防服装一批
- 空气呼吸器一批
- 灭火器一批

## 2. 产品分类规则

第一版固定使用以下分类，避免越加越乱：

- 消防救援装备
- 呼吸防护
- 消防服装
- 灭火设备
- 应急照明
- 工业安全

如遇到边界产品，优先按客户采购场景归类。例如：

- 空气呼吸器：呼吸防护
- 4 合 1 气体检测仪：工业安全
- 消防战斗服：消防服装
- 破拆工具：消防救援装备
- 应急灯具：应急照明

## 3. 每个产品建议整理成这些字段

后台产品录入字段：

- 产品分类
- 型号 / SKU
- 中文名称
- 英文名称
- 中文简介
- 英文简介
- 产品参数
- 产品图片
- 状态：上架 / 草稿 / 下架

建议展示内容：

- 产品名称：简洁清楚，不堆关键词
- 产品简介：一句话说明用途和适用场景
- 产品参数：型号、检测对象、认证、材质、容量、防护等级、适用范围等
- 产品图片：优先使用白底主图，其次使用场景图
- 价格：第一版不显示价格，统一展示“询价”

## 4. Codex 处理资料的方式

你把资料发来后，Codex 负责：

- 从 PDF / PPT / 表格里提取产品名称、型号和参数
- 从资料中挑选或裁切适合官网展示的图片
- 统一产品命名风格
- 判断产品应该放入哪个分类
- 压缩和整理图片
- 编写中文简介和英文简介
- 整理参数为适合网页阅读的格式
- 录入后台或生成可批量导入的数据
- 检查前台展示效果

## 5. 你需要确认什么

每批产品上线前，你主要确认三件事：

- 分类是否符合你的业务习惯
- 产品名称和型号是否准确
- 供应商资料是否允许公开展示

如果产品涉及品牌授权、代理权限、证书有效期，建议上线前重点确认。

## 6. 图片选择标准

优先级：

1. 清晰白底产品图
2. 官方场景图
3. 产品细节图
4. 参数页截图或宣传页局部裁切

避免使用：

- 带大量水印的图片
- 分辨率过低的图片
- 被拉伸变形的图片
- 背景杂乱且看不清产品的图片

## 7. 推荐工作流

每次新增产品按这个节奏走：

1. 你发供应商资料
2. Codex 整理产品清单和分类
3. Codex 提取图片、参数和介绍
4. Codex 给你看第一批录入效果
5. 你确认名称、分类和重点产品
6. Codex 批量补齐并上线
