# 帝昂官网设计规范

## 设计定位

帝昂官网面向消防救援装备与工业安全产品采购客户，视觉气质应保持专业、可信、清晰、采购友好，并带有克制的工业科技感。

推荐方向：

- 官网前台：`minimalist-flat` + 工业 B2B 企业风
- 后台管理：`fluent-design` + 数据心跳感
- 客户中心：`material-design` + 流程状态动效

不建议用于主站的风格：

- 赛博朋克、像素、漫画、Y2K、强玻璃拟态、强新野兽派
- 大面积霓虹、持续闪烁、复杂粒子、强娱乐化插画

## 视觉原则

1. 信息优先
   页面首先服务采购决策：产品、场景、资料、联系方式必须清楚可扫读。

2. 动效克制
   动效只用于提示层级、状态变化和引导视线，不做无意义炫技。

3. 工业可信
   使用深蓝、钢灰、白色、金色作为主基调，表达可靠、稳重和专业。

4. 留白清楚
   官网保留呼吸感；后台则更紧凑，适合重复操作和快速扫描。

5. 一致入口
   后台入口统一使用 `https://www.da-fire.com/zh/admin/login`，避免裸域名和 IP 导致登录态不稳定。

## 色彩系统

沿用当前 Tailwind 色板：

- `ink` `#0B1729`：主深色，导航、页脚、按钮、深色区块
- `navy` `#10223A`：辅助深蓝，深色卡片和背景层次
- `steel` `#526171`：正文辅助文字
- `gold` `#C99A49`：品牌强调色，按钮、标签、关键线条
- `ember` `#D63B2A`：警示、错误、风险提醒
- `mist` `#F5F7FA`：浅灰背景，分区和后台表面

使用比例建议：

- 白色/浅灰：55%-65%
- 深蓝/墨色：25%-35%
- 金色强调：5%-8%
- 红色警示：只用于异常状态

## 字体与层级

字体：

- 中文：`PingFang SC`, `Microsoft YaHei`
- 英文/数字：`Arial`

官网前台：

- 首页主标题：`text-4xl` 到 `text-6xl`，只用于首屏
- 分区标题：`text-3xl` 到 `text-4xl`
- 卡片标题：`text-lg` 到 `text-xl`
- 正文：`text-base`，行高 `leading-7` 或 `leading-8`

后台/客户中心：

- 页面标题：`text-2xl`
- 面板标题：`text-base` 到 `text-lg`
- 表格和表单：`text-sm`
- 数据数字：`text-2xl` 到 `text-4xl`，可配合轻微数字滚动

## 布局规范

官网前台：

- 首屏必须直接表达品牌、业务和主要行动入口
- 每屏只突出一个核心主题
- 产品、场景、案例、联系信息采用清晰分区
- 联系方式区域字号要柔和，邮箱、电话不应断裂或拥挤

后台管理：

- 避免营销式大 hero
- 使用稳定的顶部栏、侧边栏、数据卡、表格和筛选器
- 常用操作按钮固定位置，减少寻找成本

客户中心：

- 以项目、询盘、资料、服务进度为核心
- 状态流转应清楚：已提交、处理中、需补充、已完成

## 动效规范

整体原则：

- 动效时长：`160ms` 到 `420ms`
- 缓动：`ease-out` 或柔和 cubic-bezier
- 循环动画必须低强度、低频率
- 尊重 `prefers-reduced-motion`

官网可用动效：

- 首屏背景轻微视差或暗光扫描
- 按钮 hover 轻微上浮与颜色过渡
- 产品卡片 hover 边框变金色、阴影增强
- 数据数字进入视口时递增一次
- 流程线条在进入视口时按顺序点亮
- 联系卡片轻微 hover，不做持续跳动

后台可用动效：

- 数据卡片轻微 pulse，模拟“心跳感”
- 新询盘进入时行背景闪一下再回落
- 图表线条加载时从左到右绘制
- 状态 badge 变化时做 200ms 色彩过渡
- 操作成功后按钮或 toast 给出短反馈

不使用：

- 大面积闪烁
- 长时间旋转
- 过强弹跳
- 遮挡正文的动画
- 自动播放且不可停的复杂动效

## 组件规范

按钮：

- 主按钮使用 `gold` 背景，文字用 `ink`
- 次按钮使用透明或白色背景，加细边框
- 危险操作使用 `ember`
- 按钮圆角保持 `rounded-md`

卡片：

- 圆角不超过 8px
- 官网卡片可有轻阴影
- 后台卡片以边框和浅背景为主，减少装饰

表格：

- 表头浅灰底
- 行 hover 使用 `mist`
- 关键状态使用 badge，不用大面积色块

表单：

- 输入框高度统一
- 错误提示靠近字段
- 必填项清楚标注，但不制造压迫感

图标：

- 优先使用 `lucide-react`
- 图标线性、简洁、尺寸统一
- 功能按钮优先图标+tooltip 或图标+文字

## 官网升级建议

第一阶段：轻动效增强

- 首页首屏按钮、卡片、流程区增加 hover 和进入视口动画
- 产品卡片增加边框光感
- 联系我们区保持现在的柔和字号，不再放大

第二阶段：数据感增强

- 首页加入少量数字指标，如服务项目、产品资料、响应流程
- 数字只在进入视口时滚动一次
- 不做持续跳动，避免影响采购阅读

第三阶段：后台心跳感

- 后台首页数据卡做轻 pulse
- 询盘表格新数据做短暂高亮
- 管理首页增加小型趋势图和状态反馈

## 页面适配

桌面端：

- 以 7xl 内容宽度为主
- 保持双栏和多卡片布局
- 重要联系信息不换行时优先降低字号，而不是强行撑开卡片

移动端：

- 首屏标题适当降级
- 按钮竖向排列
- 联系卡片单列展示
- 不使用横向滚动承载核心信息

## 品牌语气

中文文案：

- 简洁、明确、偏采购服务语气
- 少用夸张词，如“顶级”“颠覆”“震撼”
- 多用可执行表达，如“提交采购需求”“资料确认”“项目跟进”

英文文案：

- 以 B2B 服务表达为主
- 避免过度营销化口号
- 重点突出 sourcing, service, documents, support

## 维护原则

后续新增页面或组件时，优先沿用：

- 当前 Tailwind 色板
- `rounded-md`
- `shadow-soft`
- `lucide-react`
- 深蓝 + 白色 + 金色强调
- 克制动效和清晰信息层级

任何新风格进入项目前，先判断是否符合“消防装备采购官网”的专业可信气质。
