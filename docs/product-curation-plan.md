# 帝昂官网产品整理与上传规范

## 目标

官网产品库先做询价展示，不追求把品牌目录全量搬上来。每个品类优先选择 5 款代表产品，突出品牌、型号、适用场景、核心卖点和询价入口。

## 品牌范围

- 3M
- 霍尼韦尔 Honeywell
- MSA 梅思安

## 统一品类

- 呼吸防护
- 眼面防护
- 听力防护
- 手部防护
- 坠落防护
- 气体检测
- 消防应急装备
- 工业安全综合

## 产品字段

- 品牌：3M / 霍尼韦尔 Honeywell / MSA 梅思安 / 其他品牌
- 产品分类：使用统一品类
- 型号/SKU：目录型号或订货号
- 中文产品名称
- 英文产品名称
- 中文简介：适用场景、核心卖点、认证信息、采购说明
- 英文简介：简洁英文展示
- 参数/备注：规格、包装、认证、供方信息、选型注意事项
- 参考价/目录价：可选，最终成交仍以报价为准
- 产品图片：优先使用清晰白底图或目录中的主图裁切图
- 状态：先草稿，确认无误后上架

## 上传原则

1. 每个品类先放 5 款代表产品，不够后续补。
2. 同一品类内尽量覆盖不同品牌，避免全是一个品牌。
3. 标准价格只作为参考价展示，不作为最终成交价。
4. 文案不直接复制长目录，改写为官网采购场景语言。
5. 产品先以草稿录入，确认图片、分类、型号无误后再上架。
6. 每次批量上传前先 `git pull`，上传后保存数据并推送，避免两台电脑内容后退。
